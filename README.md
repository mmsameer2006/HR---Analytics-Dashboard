# 📊 HR Analytics Dashboard

## 📌 Project Overview
This project is an enterprise-level HR Analytics Dashboard developed using Power BI, Microsoft Excel, Power Query, and DAX. It provides actionable insights into employee performance, workforce demographics, attrition trends, and key HR metrics through interactive visualizations.

## 🎯 Objectives
- Analyze employee attrition
- Monitor workforce KPIs
- Evaluate department performance
- Understand employee demographics
- Support HR decision making

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- Power Query
- DAX

## 📈 Dashboard Features
- Employee Overview
- Attrition Analysis
- Department-wise Analysis
- Gender Analysis
- Age Group Analysis
- KPI Cards
- Interactive Filters

## 📊 Key KPIs
- Total Employees
- Active Employees
- Attrition Rate
- Average Salary
- Average Age
